# BetterDiscord Plugins

Repository containing custom BetterDiscord plugins:

## Plugins list

1. **BetterGuildTooltip** ([BetterGuildTooltip.plugin.js](BetterGuildTooltip.plugin.js))
   - Displays both the online and total member count directly in the server tooltip when hovering over a server icon in your server list.
   - Clean, lightweight, and uses inline styling to ensure stable rendering.

2. **ServerInfo** ([ServerInfo.plugin.js](ServerInfo.plugin.js))
   - Displays a premium, rich-looking server information modal when clicking "Server Profile" (Профиль сервера) in the guild context menu.
   - Includes list of friends, blocked users, and ignored users on that server.

3. **Translator** ([Translator.plugin.js](Translator.plugin.js))
   - Allows translating incoming and outgoing messages in Discord. Fixed to support dynamic toggling without reloading Discord.

4. **SmartStatus** ([SmartStatus.plugin.js](SmartStatus.plugin.js))
   - Automatically switches your network status (Online, DND, Idle) based on Discord window focus and user activity.

5. **AnimatedStatus** ([Animated_Status.plugin.js](Animated_Status.plugin.js))
   - Animates your custom status with custom texts and emojis, cycling through steps. Fixed loading format error.
